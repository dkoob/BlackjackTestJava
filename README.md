Basic Blackjack Gameloop in Java with a UI 

Will eventually add features like splitting, cut cards, multiple deck shoes, actual start menus, maybe multiplayer, surrenders, insurance, and other game mechanics. 

This is just a fun practice project to learn Java Swing and how to create a living program loop with different states and UI...

The table background was made by me in Photoshop (not where my skills lie... I think it took me like 3 hours, and I am still not happy with it, lol) <- the wood texture for the border was taken from a random picture online
The card fronts and backs came from a spritesheet (see my spritesheet manager to understand how that was implemented) posted at https://sudo-whoami.itch.io/pixel-playing-cards by John Riggles, a.k.a. "sudo_whoami" (thank you!) <- No changes were made to any of these assets.
